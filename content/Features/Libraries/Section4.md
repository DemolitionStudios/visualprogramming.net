* 3d rendering library based on the open source <a href="https://stride3d.net/" target="_blank">Stride 3d Engine</a>
* High-level scenegraph based workflow with lights, shadows, an extensive material pipeline and various PostFX (ambient occlusion, depth of field, ...)
* Low-level shader based workflow with full access to the engine
* VR output
* Visual GPU programming, SDFs, Particle Systems provided via <a href="https://www.thefuselab.io" target="_blank">the FUSE library</a>
